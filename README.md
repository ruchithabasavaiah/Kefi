# Kefi

Production-ready e-commerce backend built with:

- Node.js
- Express
- TypeScript
- PostgreSQL
- Prisma
- Stripe Payments
- JWT Authentication
- Role-based access control

## Features

- User authentication
- Product & variant management
- Secure checkout flow
- Atomic inventory updates
- Stripe PaymentIntent integration
- Webhook-based order confirmation

## Tech Stack

Node.js · Express · Prisma · PostgreSQL · Stripe · Zod · JWT
